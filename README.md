# terraform-aws-security-baseline

AWS Organizations 環境におけるセキュリティサービスの Terraform 構築レシピ集です。

## Recipes

| レシピ | 概要 |
|---|---|
| [GuardDuty 全リージョン一元管理](guardduty-multiregion-setup.md) | GuardDuty を全リージョンで有効化し、単一 Terraform ワークスペースから管理。MEDIUM+ Finding のメール通知付き |

## 使い方

各レシピは AI アシスタントへの構築プロンプトとして使えます。前提条件のパラメータを埋めてプロンプトとして渡すことで、Terraform コードを生成できます。
